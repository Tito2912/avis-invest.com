FAQ

eToro est-il régulé ?

eToro opère via plusieurs entités régulées (UE : CySEC, UK : FCA, AU : ASIC). Votre entité dépend de votre pays de résidence.

Les actions sont-elles vraiment à 0 % de commission ?

Oui pour les actions réelles (hors CFD). D'autres coûts peuvent exister (conversion, spreads, etc.).

Puis-je copier d'autres investisseurs ?

Oui via CopyTrader™. La copie n'est pas une garantie de performance ; vous pouvez perdre de l'argent.

Quels sont les principaux risques ?

La valeur de vos investissements peut baisser comme augmenter. Les CFD comportent un risque élevé : 61 % des comptes de détail perdent de l'argent chez ce fournisseur.
